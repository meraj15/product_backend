# Product Admin

## Overview
Product Admin is a Flutter application that allows admin users to manage products, add/update new products, view orders, and manage reviews.

## Installation

### 1. Clone the Repository:
```sh
git clone https://github.com/meraj15/product_backend.git
```

### 2. Navigate to the Project Directory:
```sh
cd Product-Admin
```

### 3. Install Dependencies:
```sh
flutter pub get
```

### 4. Run the Application:
```sh
flutter run
```

## Features

- **Product Management:** Add, update, and delete products.
- **Order Management:** View and manage order statuses.
- **Reviews Handling:** View and manage customer reviews.
- **Dashboard:** An intuitive dashboard displaying all important metrics.
